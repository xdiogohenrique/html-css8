Exemplo básico utilizando a regra @media



    Veja a utilização da regra @media: 
        .coluna1,
        .coluna2,
        .coluna3 {
            height: 300px;
        }

        /*
           observe que para telas menores de 320px de largura
           não existe uma especificação, então as divs terão comportamento
           padrão, ocupando a largura do seu contêiner (body no exemplo)
        */
        
        @media screen and (min-width: 320px) {
            /* para telas a partir de 320px de largura */
            .coluna1 {
                background-color: green;
            }

            .coluna2 {
                background-color: lime;
            }

            .coluna3 {
                background-color: olive;
            }
        }

        @media screen and (max-width: 780px) {
            /* para telas até 780px de largura */
            .coluna1,
            .coluna2 {
                float: left;
                width: 50%;
            }

            .coluna3 {
                clear: both;
            }
        }
        
        /*
           observe que na faixa entre 780px e 1024px de largura
           não existe uma especificação, então as divs terão comportamento
           padrão, ocupando a largura do seu contêiner (body no exemplo)
        */
        
        @media screen and (min-width: 1024px) {
            /* para telas a partir de 1024px de largura */
            .coluna1 {
                float: left;
                width: 30%;
            }
            .coluna2 {
                float: left;
                width: 60%;
            }
            .coluna3 {
                float: left;
                width: 10%;
                clear: none;
            }
        }
